The first input is to calculate the xp needed to level up once at the inputted level.

The second input is to calculate the xp needed to get to the specified level from 0.

The third input is to calculate the xp needed to get from the level in the first field to the level in the second field.

All inputs should be integers, if they are not they will be rounded down (except the multiplier).

Levelz does allow negative parameters but I do not, because they cause issues and if you have negative parameters something is wrong with you (they will get set to 0).


The equation parameters (will default to Valhelsia: Enhanced Vanilla parameters if blank).






Current equation is:

Floor(level2*0.07+50)

With a maximum output of 10000 XP and a maximum input of level 479.
